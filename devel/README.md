This directory "devel" is intended for draft or test code and other
stuff which is not suitable for "normal" users. Its contents is
ignored during the creation of the release tarball.

The subdirectories contain the following

|  Directory |  Purpose                                     |
|:-----------|:---------------------------------------------|
| `bugs`     | m-files realted to open(?) bugs              |
| `doc`      | Files for creating the pdf-/qch-documentation|
| `exapmples`| m-files with some examples                   |
| `release`  | Checklists for preparing a package release   |
| `tests`    | Tests scripts                                |
| `utils`    | Some utility scripts                         |
| `wip`      | Work in progress of some new features        |
